## Quick Examples

<CodeGroup>

```bash Claude Code
# Start Claude Code with dashboard integration
omnara
```

```bash Codex
# Start Codex with custom name
omnara --agent codex --name "Backend Refactor"
```

```bash Headless
# Run Claude Code in background with auto-accept
omnara headless --permission-mode acceptEdits
```

```bash Webhook Server
# Start webhook server with public URL
omnara serve
```

</CodeGroup>

## Global Options

These options work across all commands:

| Option | Description |
|--------|-------------|
| `--api-key <key>` | API key for authentication |
| `--auth` | Authenticate or re-authenticate |
| `--reauth` | Force re-authentication |
| `--version` | Show version information |
| `--base-url <url>` | Custom API server URL |
