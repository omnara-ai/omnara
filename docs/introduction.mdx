## What is Omnara?

Omnara transforms your AI agents (Claude Code, Codex CLI, n8n, and more) from silent workers into communicative teammates. Get real-time visibility into what your agents are doing, and respond to their questions instantly from a single dashboard on web and mobile.

## Key Features

<AccordionGroup>
  <Accordion title="Real-time Dashboard" icon="gauge">
    See exactly what your AI agents are doing in real-time across terminal, web, and mobile. Never lose track of your agents again.
  </Accordion>
  <Accordion title="Two-way Communication" icon="messages">
    Your agents can ask questions and wait for your response. Reply from anywhere - your phone, web dashboard, or terminal.
  </Accordion>
  <Accordion title="Multi-agent Support" icon="robot">
    Works with Claude Code, Codex CLI, n8n workflows, and any custom agent via our SDK or REST API.
  </Accordion>
  <Accordion title="Session Persistence" icon="database">
    All messages and sessions are stored securely. Resume conversations, review history, and track progress over time.
  </Accordion>
</AccordionGroup>

## Quick Start

Get started in under a minute:

<CodeGroup>

```bash pip
pip install omnara
omnara
```

```bash homebrew
brew install omnara
omnara
```

```bash uv
uv tool install omnara
omnara
```

</CodeGroup>

That's it! Create an account when prompted, then interact with Claude in your terminal. Everything syncs automatically to your [dashboard](https://omnara.com/dashboard).
